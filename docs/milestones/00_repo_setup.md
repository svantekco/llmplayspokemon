# Milestone 00 — Repo, environment, and development harness

## Goal
Create a clean local project that can run in mock mode immediately.

## Outcomes
- Python project boots locally
- dependencies install cleanly
- package structure exists
- mock mode runs without emulator
- logging works
- tests run

## Definition of done
- `python -m pokemon_agent.main --mode mock` runs
- `pytest` runs
- config loads from `.env`

## Notes
Do not start with LLM work before this is stable.
