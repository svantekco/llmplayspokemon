# Milestone 08 — Hardening, evaluation, and recovery

## Goal
Make the system robust enough for longer unattended runs.

## Outcomes
- save/load recovery flow
- better no-progress recovery heuristics
- token usage tracking
- prompt compactness review
- scenario-based evaluation

## Definition of done
- agent can recover from common failure modes more gracefully
- you can compare model quality and cost across runs
